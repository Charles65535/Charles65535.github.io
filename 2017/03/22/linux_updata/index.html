linux内核升级

发表于 2017-03-22 | 分类于 运维

1.下载内核 从以下网站https://www.kernel.org/

2.上传kernel文件到/usr/src/kernels/ 目录下，此处也可以选择其他目录

3.解压kernel文件

* [root@apache kernels]# tar -xJvf linux-3.16.1.tar.xz

4.安装包 gcc、ncurses-devel

* [root@apache yum.repos.d]# yum -y install gcc ncurses-devel

6.执行 make menuconfig命令 出现图形

7.save

8.生成.config文件

9.退出

10.提示执行make继续或查看帮助

11.执行 make命令，根据环境编译kernel，生成模块文件。等待一段时间

13.执行 make modules_install 安装模块

14.执行 make install

15.重启shutdown -r 0 选择新的kernel启动

16.查看当前kernel，uname -r，升级内核完成
