数字图像处理-对比度调整

发表于 2017-03-23 | 分类于 编程

原理

　　图像最亮和最暗之间的区域之间的比率，比值越大，从黑到白的渐变层次就越多，从而色彩表现越丰富。对比度对视觉效果的影响非常关键，一般来说对比度越大，图像越清晰醒目，色彩也越鲜明艳丽；而对比度小，则会让整个画面都灰蒙蒙的。对比度处理时将R，G，B三个值同时乘以一个比例，就可以得到对比度变化的效果。

　　本程序公式如下：
