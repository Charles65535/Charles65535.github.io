数字图像处理-阈值分割

发表于 2017-03-23

##原理 ##
　　图像阈值化的目的是要按照灰度级，对像素集合进行一个划分，得到的每个子集形成一个与现实景物相对应的区域，各个区域内部具有一致的属性，而相邻区域不具有这种一致属性。这样的划分可以通过从灰度级出发选取一个或多个阈值来实现。

阈值分割原理：

　　一副图像包括目标、背景和噪声，设定某一阈值T将图像分成两部分：大于T的像素群和小于T的像素群。

　　在实际处理时候，为了显示需要一般用255（白色）表示背景，用0（黑色）表示对象物。
图像灰度化的算法比较常见的有Otsu、最大熵、迭代法、自适应阈值、手动、迭代法、基本全局阈值法等。本文主要介绍下以下几种：

1) 双峰法：
直方图双峰法是直接从图像的灰度分布直方图上来确定阈值。一般选取双峰之间的谷底处像素灰度值为阈值。该方法不适用于单峰或多峰直方图的情况。

2) 迭代法：
迭代阈值法先选择一个近似阈值作为估计值的初始值,然后不断改进这一估计值。阈值的改进策略是关键,初始阈值为图像的平均灰度T0,用T0将图像的象素点分作两部分,计算各自平均灰度,小于T0的部分平均灰度为Tlow,大于T0的部分平均灰度为Thigh。计算T1=(Thigh+Tlow)/2将T1作为新的全局阈值代替T0,重复以上过程,如此迭代,直至TK收敛。即TK+1=TK,也就是Thigh,Tlow的均值不再变化。

3) 自适应：
自适应阈值方法也叫动态阈值方法,针对图像中有阴影、光照不均匀、各处的对比度不同、突发噪声等情况,整幅图像分割将没有合适的单一门限,因为单一的阈值不能兼顾图像各个像素的实际情况。这时,可以对图像按照坐标分块,对每一块分别选一阈值进行分割。

　　自适应阈值的常用方法是对每个像素确定以它为中心的邻域窗口,计算窗口内像素的最大值和最小值,然后取它们的均值作为阈值。

##步骤 ##
本次程序采用第二种方法即迭代法，使用Python语言实现。具体如下：

  1. 导入图像并转化为并转化为numpy的数据对象；将图像灰度化预处理；
  2. 设置初始阈值为最大灰度值和最小灰度值的平均值；初始化区域high和low以及计数器；设置两次阈值差初始值为3；
  3. 用迭代法计算分割阈值：
    1） 循环遍历每个位置上的像素点,判断该点的灰度值与阈值的关系,如果大于等于阈值将该灰度值加到high中并将计数器count_high加1,如果小于阈值将该灰度值加到low中并将计数器count_low加1；
    2） 通过将high除以计数器得出大于等于阈值区域的平均灰度值；
    3） 通过将low除以计数器得出小于阈值区域的平均灰度值；
    4） 将区域high和区域low的平均灰度相加并除以2得出新的阈值；
    5） 计算新阈值和老的阈值的差值；
    6） 重置大小灰度区域和计数器；
    7） 迭代计算，直到阈值差值为0；
  4. 循环遍历每个位置上的像素点,将大于等于阈值点灰度值设置为255，小于阈值点灰度值设置为0；
  5. 通过matplotlib模块将修改的后的图像输出显示。
    ##处理结果 ##

代码

# !/usr/bin/env python
# -*- coding:utf-8 -*-
from PIL import Image
import matplotlib.pyplot as plt
import numpy as np
import gray


def find_value(image):
    """find_value 函数用于阈值分割寻找合适的阈值"""
    threshold_value = (int(image.min()) + int(image.max())) / 2  # 设置初始阈值为最大灰度值和最小灰度值的平均值
    print("初始阈值: %d" % threshold_value)
    high = 0  # 大于阈值的区域的平均灰度,初始值为0
    low = 0  # 小于阈值的区域的平均灰度,初始值为0
    count_high = 0  # 计数器，大于阈值区域,初始值为0
    count_low = 0  # 计数器，小于阈值区域,初始值为0
    value_diff = 3  # 两次阈值差初始值为3
    while abs(value_diff) > 0:
        for i in range(0, image.shape[0]):
            for j in range(0, image.shape[1]):
                if im[i, j, 0] >= threshold_value:  # 大于阈值，将该灰度值加入high区域中
                    high = im[i, j, 0] + high
                    count_high += 1
                else:  # 小于阈值，将该灰度值加入low区域中
                    low = im[i, j, 0] + low
                    count_low += 1

        high = high / count_high  # high区域平均灰度值
        low = low / count_low  # low区域平均灰度值

        value_diff = (high + low) / 2 - threshold_value  # 新差值
        threshold_value = (high + low) / 2  # 新阈值
        print("新阈值: %f" % threshold_value)
        print("阈值差值：%f" % value_diff)
        high = 0  # 重置
        low = 0
        count_high = 0
        count_low = 0

    return threshold_value


def segmentation(image, threshold_value):
    """segmentation 函数用于对图像进行阈值分割"""
    for i in range(0, image.shape[0]):
        for j in range(0, image.shape[1]):
            if image[i, j, 0] >= threshold_value:
                image[i, j, 0] = 255
                image[i, j, 1] = 255
                image[i, j, 2] = 255
            else:
                image[i, j, 0] = 0
                image[i, j, 1] = 0
                image[i, j, 2] = 0


if __name__ == '__main__':
    im = np.array(Image.open('lena.jpg'))
    gray.graying(im)  # 灰度化预处理
    value = find_value(im)
    segmentation(im, value)
    plt.imshow(im)
    plt.show()
